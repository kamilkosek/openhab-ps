# Remove-OHRule

## SYNOPSIS
Removes an existing rule corresponding to the given UID.

## SYNTAX

### Delete (Default)
```
Remove-OHRule -RuleUid <String> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-OHRule -InputObject <IOpenHabRestApiIdentity> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Removes an existing rule corresponding to the given UID.
