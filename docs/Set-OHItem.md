# Set-OHItem

## SYNOPSIS
Adds a new item to the registry or updates the existing item.

## SYNTAX

### UpdateExpanded (Default)
```
Set-OHItem -Itemname <String> [-AcceptLanguage <String>] [-Category <String>] [-FunctionName <String>]
 [-FunctionParams <String[]>] [-GroupNames <String[]>] [-GroupType <String>] [-Label <String>]
 [-Name <String>] [-Tags <String[]>] [-Type <String>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Update
```
Set-OHItem -Itemname <String> -Body <IGroupItemDto> [-AcceptLanguage <String>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentity
```
Set-OHItem -InputObject <IOpenHabRestApiIdentity> -Body <IGroupItemDto> [-AcceptLanguage <String>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Set-OHItem -InputObject <IOpenHabRestApiIdentity> [-AcceptLanguage <String>] [-Category <String>]
 [-FunctionName <String>] [-FunctionParams <String[]>] [-GroupNames <String[]>] [-GroupType <String>]
 [-Label <String>] [-Name <String>] [-Tags <String[]>] [-Type <String>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Adds a new item to the registry or updates the existing item.
